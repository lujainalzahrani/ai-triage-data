# AI Triage Data Samples (TRL4)

هذا المستودع يحتوي عينات بيانات لنظام **AI Triage System** المستخدم في هذا المشروع، بهدف توضيح كيفية عمل النظام في مستوى TRL4 على بيانات حقيقية أو شبه حقيقية.

## محتويات البيانات

جميع الملفات موجودة داخل مجلد [`data`](./data):

- `patients_sample.csv`  
  يحتوي بيانات تعريفية مبسطة عن المرضى، مثل:
  - رقم الهوية (id_number)
  - الاسم (name)
  - العمر (age)
  - وجود أمراض مزمنة (has_chronic)
  - وجود أعراض مقلقة حالية (has_severe_symptoms)

- `visits_sample.csv`  
  يحتوي عينات من زيارات المرضى للنظام، مثل:
  - وقت الزيارة (visit_time)
  - مصدر القراءة (source) مثل الكاميرا
  - عمر المريض وقت الزيارة (age)
  - هل العمر 65+ (is_65_plus)
  - أولوية الحالة (priority)
  - رقم المريض/الكود (patient_code)

- `triage_questions.csv`  
  يحتوي أسئلة الترياج المستخدمة في النظام، مع خيارات الإجابة والأوزان لكل إجابة للمساعدة في تقييم شدة الحالة.

- `triage_logic_samples.csv.csv`  
  يحتوي أمثلة على منطق الفرز (triage logic) والنتيجة النهائية لكل مريض، مثل:
  - العمر التقديري
  - وجود أمراض مزمنة أو أعراض مقلقة
  - تفسير سبب أولوية الحالة (final_priority_reason)

## الغرض من البيانات

- توضيح كيفية تمثيل بيانات المرضى والزيارات في نظام AI Triage.
- دعم توثيق مستوى TRL4 من خلال أمثلة واقعية لطريقة التخزين والفرز.
- تم تبسيط البيانات وإخفاء أي معلومات حقيقية حساسة؛ الأرقام والأسماء المستخدمة لأغراض تعليمية وتجريبية فقط.
